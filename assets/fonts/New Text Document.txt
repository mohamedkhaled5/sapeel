Introduction
Base URL: https://api.quranpedia.net
This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).
Authenticating requests
This API is not authenticated.

Search
Search for items of a specified type using a query.

Example request:
$client = new \GuzzleHttp\Client();
$url = 'https://api.quranpedia.net/v1/search/سورة/';
$response = $client->get(
    $url,
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));

Request    Try it out ⚡   
GET v1/search/{query}/{type?}

Headers
Content-Type      
Example: application/json

Accept      
Example: application/json

URL Parameters
query   string   
The search query Example: سورة

type   string  optional  
The type of items to search for !! Available types : notes|fatwas|topics|books

Mushafs
Get All Mushafs

Each JSON representation of a mushaf includes an 'images' property, which contains a compressed zip file. This zip file contains SVG and JSON files for the Surahs.

Example request:
$client = new \GuzzleHttp\Client();
$url = 'https://api.quranpedia.net/v1/mushafs';
$response = $client->get(
    $url,
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
